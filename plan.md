# LLM Self vs Cross Critique Research Plan

## 1. Research Goal
This project studies how LLMs critique solutions depending on identity attribution.

Main question:
Do models critique solutions differently when they believe the solution is:
- Their own solution
- Another model's solution
- Anonymous solution
- Human solution

Key hypothesis:
Self-critique failure is partially caused by identity attribution and context anchoring, not only reasoning limitations.

## 4. Experiment Pipeline

### Step 1 — Generate Solutions
Multiple solver models generate solutions for each problem.

Create a solution pool:
(problem, solver_model, solution, ground_truth_correct)

### Step 2 — Sample Solutions
Split solutions into:
- Correct solutions
- Wrong solutions

Sample balanced sets:
- N correct
- N wrong

### Step 3 — Critique Conditions
Each solution is evaluated under different identity conditions:
1. **Self**: "This is your solution."
2. **Other model**: "This solution was generated by another AI model."
3. **Anonymous**: "This is a solution."
4. **Human**: "This solution was written by a human student."

For each condition, the critic must:
- Detect whether the solution is correct
- Explain errors
- Revise the solution if needed
- Provide confidence

---

## 5. Main Metrics

### Detection Accuracy
Whether the critic correctly identifies if the solution is correct or wrong.

### Correction Success Rate
Among wrong solutions: Wrong → Correct

### Harmful Correction Rate
Among correct solutions: Correct → Wrong

### False Error Rate
Correct solutions labeled as wrong.

### Self vs Cross Gap
Difference between performance when evaluating own solution vs others.

---

## 6. Main Experiments

### Experiment 1 — Identity Swap (Main)
Same solution, different identity labels:
- Self / Other model / Anonymous / Human

Measure:
- Detection accuracy
- Correction success
- Harmful correction

### Experiment 2 — Solver–Critic Matrix
Matrix of solver vs critic:
Each model evaluates its own and others' solutions.
Reveals cross-model critique patterns.

### Experiment 3 — Session / Anchoring Ablation (Optional)
- Same conversation
- New conversation
- Paraphrased solution

---

## 7. Key Result to Show
The most important comparison:
Self critique vs Cross critique performance difference.

Main claim:
Models are worse at detecting and correcting errors in their own solutions than in solutions attributed to other sources.
